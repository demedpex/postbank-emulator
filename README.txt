우체국 스마트뱅킹 에뮬레이터 배포판
========================================

📦 배포 방법
-----------

1. ZIP 파일 압축 해제
   - 모든 파일을 같은 폴더에 유지하세요

2. 배포 옵션

   [방법 1] 웹 서버 사용 (권장)
   - 모든 파일을 웹 서버에 업로드
   - emulator.html을 브라우저로 열기
   - SVG 파일들이 같은 폴더에 있어야 합니다

   [방법 2] GitHub Pages
   - 모든 파일을 GitHub 저장소에 업로드
   - Settings > Pages에서 배포
   - 자동으로 HTTPS URL 제공

   [방법 3] 로컬 실행
   - 모든 파일을 같은 폴더에 두고
   - emulator.html을 더블클릭
   - (Chrome에서 CORS 오류 발생 시 --allow-file-access-from-files 옵션 사용)


📁 필수 파일
-----------
- emulator.html (메인 파일)
- 1.svg ~ 13.svg (화면 이미지)
- 6-1.svg, 7-1.svg (특정 화면)


⚠️ 주의사항
-----------
1. 모든 SVG 파일이 HTML과 같은 폴더에 있어야 합니다
2. 파일명을 변경하지 마세요
3. 인터넷 연결이 필요합니다 (Pretendard 폰트 로드)


🔊 TTS 기능
-----------
- 브라우저 자동재생 정책으로 인해 첫 실행 시 TTS가 작동하지 않을 수 있습니다
- TTS 토글 버튼을 한 번 클릭하면 정상 작동합니다
