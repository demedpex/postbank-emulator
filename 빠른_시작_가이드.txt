📌 가장 쉬운 방법 (GitHub Pages)
═══════════════════════════════════════════════

⏱️ 소요시간: 약 3~5분
💰 비용: 무료
🔒 요구사항: GitHub 계정 (무료)


🚀 5단계로 끝내기
─────────────────────────────────────────────

1️⃣  GitHub 가입 & 로그인
   → https://github.com
   → 이미 계정이 있다면 로그인


2️⃣  새 저장소 만들기
   → 오른쪽 상단 [+] 버튼
   → New repository 클릭
   → Repository name: post-banking-emulator
   → Public 선택
   → Create repository 클릭


3️⃣  파일 업로드
   → "uploading an existing file" 클릭
   → 아래 파일들 드래그 앤 드롭:

     ✓ emulator.html
     ✓ 1.svg ~ 13.svg (13개)
     ✓ 6-1.svg, 7-1.svg (2개)

   → Commit changes 클릭


4️⃣  GitHub Pages 활성화
   → Settings 메뉴 클릭
   → 왼쪽 Pages 클릭
   → Branch: main 선택
   → Save 클릭


5️⃣  배포 완료! 🎉
   → 1~3분 기다리기
   → 페이지 새로고침 (F5)
   → 초록색 박스에 URL 표시됨

   예시 URL:
   https://본인아이디.github.io/post-banking-emulator/emulator.html

   ⚠️ URL 끝에 /emulator.html 꼭 붙이기!

🔧 문제 해결
═══════════════════════════════════════════════

Q. 이미지가 안 보여요 (엑박)
A. SVG 파일이 누락됨 → 모든 SVG 파일 업로드 확인

Q. 404 오류가 나요
A. URL 끝에 /emulator.html 추가

Q. TTS 소리가 안 나요
A. 페이지 로드 후 TTS 토글 버튼 한 번 클릭

Q. GitHub Pages가 활성화 안 돼요
A. 저장소를 Public으로 설정했는지 확인
